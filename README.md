# Spotify-Youtube-Demo

## Environment Setup

### Backend

#### Setup Virtual Environment
```
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

#### Run Server (http://127.0.0.1:5000)
```
cd backend
source venv/bin/activate
python3 app.py
```

### Frontend

#### Install Frameworks
```
cd frontend
yarn
```

#### Run Frontend (http://127.0.0.1:3000)
```
cd frontend
yarn start
```
